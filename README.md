# Front-end — Sistema de Alocação Acadêmica

Aplicação web em Next.js (App Router) que consome a API do backend para gerir turmas, salas, disciplinas, alocações e reservas.

## Como rodar

1. Instale dependências:

```bash
npm install
```

2. Configure variáveis de ambiente (`.env.local`):

```env
NEXT_PUBLIC_API_URL=http://localhost:3333
```

3. Suba o servidor:

```bash
npm run dev
```

Acesse: http://localhost:3000
